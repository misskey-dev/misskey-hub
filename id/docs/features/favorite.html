# Favorit

Fitur ini memperbolehkanmu untuk menandai sebuah catatan sebagai favorit. Kamu dapat melihat catatan yang kamu favoritkan pada halaman Favorit. Orang lain tidak akan diberitahu apabila kamu telah menambahkan catatan mereka ke dalam favoritmu, dan hanya kamu saja yang dapat melihat favoritmu.

Untuk memfavoritkan catatan, tekan "Tambahkan ke favorit" pada menu catatan. Untuk menonfavoritkan catatan, klik "Hapus dari favorit" pada menu catatan.
