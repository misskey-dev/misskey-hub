# Elenco dei canali

Canale in cui è convogliato il flusso di Note della Timeline Home

Canale in cui è convogliato il flusso di Note della Timeline Sociale
localTimeline
Canale in cui è convogliato il flusso di Note della timeline locale
main
Questo canale fornisce informazioni di base
