# About Misskey Hub

Misskey Hub is Misskey's official website, which provides documentation, a documentation blog, a plugin store and more. We aim to be the place for "everything Misskey".

Misskey Hub is open sourceopen in new window.

Misskey Hub is a static site built with VuePressopen in new window.
