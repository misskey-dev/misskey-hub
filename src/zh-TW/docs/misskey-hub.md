# 關於Misskey Hub
Misskey Hub是Misskey的官方網站，提供與Misskey相關的知識文件、開發部落格、外掛商店等。
目標是「Misskey的全部都在這裡」。

Misskey Hub也是[以開放原始碼發布的。](https://github.com/misskey-dev/misskey-hub)

::: tip
Misskey Hub是一個使用[VuePress](https://github.com/vuepress/vuepress-next)建立的靜態網站。
:::
