# 關於 Misskey Hub
Misskey Hub 是 Misskey 的官方網站，提供與 Misskey 相關的知識文件、開發部落格、外掛商店等。
目標是「Misskey 的全部都在這裡」。

Misskey Hub 也是[以開放原始碼發布的。](https://github.com/misskey-dev/misskey-hub)

::: tip
Misskey Hub 是一個使用 [VuePress](https://github.com/vuepress/vuepress-next) 建立的靜態網站。
:::
